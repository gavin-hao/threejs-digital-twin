# git-lfs

模型文件超过100M 采用 git-lfs([安装 Git Large File Storage](https://docs.github.com/zh/repositories/working-with-files/managing-large-files/installing-git-large-file-storage)) 存储

#### 配置 Git Large File Storage

```bash

git lfs track "*.glb"

```
